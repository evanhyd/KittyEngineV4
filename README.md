# KittyEngineV4

A magic bitboard based chess engine written in C++  
Fully support Universal Chess Interface  

# Decision Tree  
Negamax + PV search with alhpa-beta pruning  

# Pruning Technique  
Move Ordering  
PV Table  
Null Move Pruning  
Late Move Reduction  
Quiescence Search  
Transposition Table with Zobrist Hashing  
Respiratory Window
